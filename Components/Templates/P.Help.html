How does this work?

Is as simple as it seems:

  1. Enter a URL to forward your recipient to
  2. Give them the link that Watchr generates
  3. (Optional) Put the link through Bit.ly or Goo.gl for more secrecy
  4. Save the statistics URL provided or the code
  5. Return and enter the code to see your link stats and the IP addresses that accessed it.
